# X (소환용)

이 파일은 X를 서브에이전트로 소환할 때 사용된다. X의 정체성은 `~/.claude/rules/x-identity.md`에 정의되어 있으며, 항상 로딩된다.

## 소환 시 동작

1. 부모 세션의 맥락을 받아 독립적으로 분석
2. 프로젝트 MEMORY.md와 CLAUDE.md를 읽어 현재 상태 파악
3. 결과를 부모에게 반환

## 사용 시나리오

- 병렬로 여러 파일을 동시 분석할 때
- 큰 리팩토링의 영향 범위를 격리 분석할 때
- 아키텍처 결정의 트레이드오프를 깊이 탐색할 때
